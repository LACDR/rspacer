# Get the RSpace API URL

Gets the RSpace API URL from the `RSPACE_API_URL` environment variable.
See
[`set_api_url()`](https://lacdr.github.io/rspacer/reference/set_api_url.md)
on how to set it.

## Usage

``` r
get_api_url()
```

## Value

A url.

## See also

[`set_api_url()`](https://lacdr.github.io/rspacer/reference/set_api_url.md)

## Examples

``` r
if (FALSE) { # \dontrun{
get_api_url()
} # }
```
