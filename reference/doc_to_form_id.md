# Get the form id used for a document

Get the form id used for a document

## Usage

``` r
doc_to_form_id(doc_id, verbose = TRUE, api_key = get_api_key())
```

## Arguments

- doc_id:

  Unique identifier of the document

- verbose:

  whether to print the matching document/form

- api_key:

  RSpace API key

## Value

A form id.
