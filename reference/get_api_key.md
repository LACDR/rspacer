# Get the RSpace API key

Gets the RSpace API key from the `RSPACE_API_KEY` environment variable.
See
[`set_api_key()`](https://lacdr.github.io/rspacer/reference/set_api_key.md)
on how to set it.

## Usage

``` r
get_api_key()
```

## Value

An API key

## See also

[`set_api_key()`](https://lacdr.github.io/rspacer/reference/set_api_key.md)

## Examples

``` r
if (FALSE) { # \dontrun{
get_api_key()
} # }
```
