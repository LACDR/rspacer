# Function to check availability of the API service

Function to check availability of the API service

## Usage

``` r
api_status(api_key = get_api_key())
```

## Arguments

- api_key:

  RSpace API key

## Value

If the request succeeds, a list with the status message and the RSpace
version of the API.

## Examples

``` r
if (FALSE) { # \dontrun{
api_status()
} # }
```
