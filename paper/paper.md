# Implementation

rspacer is an R package and uses the httr2 package [@Wickham:2025] to wrap a subset of the RSpace API documented at <https://community.researchspace.com/public/apiDocs>.

![Functionality of the rspacer package. (A) Functions in the rspacer package include interactions with folders, documents and files through the RSpace API. (B) A user can retrieve information on folder structures using the rspacer browse and find functionality. (C) An example workflow where rspacer can be used to retrieve metadata from the RSpace ELN into R. (D) Screenshots of an example workflow where rspacer uses Quarto HTML reports or tabular assay data (CSV/TSV/XLSX) as input to create new assays in RSpace.](./figures/rspacer_functionality_v5.svg){#functionality}
